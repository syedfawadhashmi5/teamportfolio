CSS 3

To style a web page layout in HTML5 with CSS3, designers must first define the structure of the page using HTML5 elements such as divs, sections, and containers. Once the HTML5 structure is in place, CSS3 can be used to define the layout, typography, color, and other visual aspects of the page.
